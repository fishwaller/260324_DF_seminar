📹 CCTV 셋톱박스 포렌식

CCTV 영상은 수사의 핵심 증거입니다. 포렌식팀에 의뢰하면 어떤 분석이 가능한지, 그리고 현장에서 어떻게 확보해야 하는지를 안내합니다.

포렌식팀에 의뢰하면 이런 분석이 가능합니다
🕐

타임라인 재구성

녹화 메타데이터로 사건 시간대를 특정합니다. 근로시간·재해 발생 시각 입증에 활용됩니다.

🔍

영상 화질 보정

어둡거나 흐린 영상의 노출·대비·선명도를 조정하고, 누락된 프레임을 보간합니다. 일단 영상이 있으면 의뢰해 보세요.

🛡️

위변조 탐지

편집 흔적, 타임스탬프 불일치를 감지합니다. 사업주 제출 영상의 신뢰성 검증에 쓰입니다.

📼

삭제 영상 복구

덮어쓰기 전 영역에서 삭제된 프레임 복원을 시도합니다. 단, 시간이 지날수록 가능성이 낮아집니다.

골든타임 — 복구보다 확보가 먼저입니다

같은 CCTV, 다른 결과

아래 두 영상을 비교해 보세요. 왼쪽은 덮어쓰기 후 복구한 영상, 오른쪽은 골든타임 내 정상 획득한 영상입니다.

❌ 복구 영상 (프레임 짜깁기)

✅ 정상 획득 영상

복구된 영상은 프레임이 뒤섞이고 화질이 심하게 열화되어 증거 가치가 크게 떨어집니다. CCTV는 용량이 차면 오래된 영상부터 자동으로 덮어쓰므로, 영상이 필요하다면 즉시 확보하거나 포렌식팀에 의뢰하세요. 복구는 최후의 수단입니다.

현장 획득 시 주의사항
1

PC·웹 방식으로 영상을 받으면 안 되는 이유

CCTV 웹뷰어나 PC 소프트웨어로 장기간 영상을 다운로드하면 세션이 중간에 끊기거나, 네트워크 전송량 제한으로 수신이 중단됩니다. 장시간·대용량 영상은 이 방식으로 완전히 확보하기 어렵습니다.

세션 끊김 위험 전송량 제한
2

USB로 직접 복사하는 것이 가장 안전

CCTV 본체의 USB 포트에 저장장치를 직접 연결해 복사하는 방식이 세션 문제 없이 대용량을 안정적으로 확보할 수 있는 방법입니다. 단, 저장장치의 파일시스템을 반드시 확인해야 합니다.

직접 연결 권장
3

외장하드 연결 시 전력 문제

CCTV 셋톱박스의 USB 포트는 전력 공급이 제한적입니다. 외장하드는 전력 소모가 크기 때문에 인식이 안 되거나 작동 중 전원이 꺼질 수 있습니다. 별도 전원 어댑터가 있는 외장하드를 준비하거나, 큰 용량의 USB 메모리를 여러 개 활용하는 것이 현실적입니다.

전원 어댑터 별도 필요 USB 메모리 분할 활용
파일시스템 호환 문제 (체험 실습)
실험

왜 USB는 인식되고 외장하드는 안 될까?

준비된 두 저장장치를 CCTV에 직접 꽂아보세요.

USB (MBR + FAT32) → ✅ 인식됨
외장하드 (GPT + NTFS) → ❌ 인식 안 됨

CCTV 같은 임베디드 장비는 오래된 펌웨어를 사용하는 경우가 많아 FAT32 포맷만 읽을 수 있습니다. 현장 출동 전 USB는 반드시 FAT32로 포맷해서 준비하세요.

FAT32 포맷 USB 준비 필수
데이터 복구가 필요한 경우
!

영상이 이미 덮어쓰여졌거나 삭제된 경우

CCTV는 저장 용량이 꽉 차면 오래된 영상부터 자동으로 덮어씁니다. 이 경우 일반적인 방법으로는 영상을 확보할 수 없습니다. 복구를 원한다면 셋톱박스 원본을 반출해야 합니다.

시간이 지날수록 복구 가능성 감소
!

셋톱박스 원본 반출 및 분해 복구

포렌식팀이 셋톱박스를 수령한 뒤 내부 HDD 또는 낸드 플래시를 분리해 전문 장비로 이미지를 획득한 뒤 삭제된 데이터 복구를 시도합니다. 현장에서 임의로 셋톱박스를 분해하거나 초기화하면 복구 가능성이 크게 떨어집니다.

현장 분해·초기화 금지 원본 그대로 반출 요청
핵심 요약
⏰

골든타임이 생명

복구보다 확보가 먼저. 영상이 필요하면 즉시 행동하세요.

💾

FAT32 USB를 현장에

CCTV에 꽂을 USB는 반드시 FAT32로 포맷해서 준비하세요.

⚡

외장하드는 전력 문제

전원 어댑터 없는 외장하드는 CCTV 포트에서 작동하지 않을 수 있습니다.

📞

어려우면 포렌식팀에

획득이 어렵거나 이미 삭제된 경우, 원본을 그대로 포렌식팀에 넘기세요.

확인 퀴즈
Q. CCTV 영상을 확보하기 위해 웹뷰어로 장시간 다운로드를 시도했더니 중간에 멈춰버렸습니다. 가장 적절한 대처는?

⚠ 영상이 필요한 경우 가능한 한 빨리 확보하거나 의뢰하세요. CCTV는 용량이 차면 자동으로 오래된 영상을 덮어쓰기 때문에 시간이 지날수록 복구 가능성이 낮아집니다.

📱 모바일 포렌식

노동사건에서 휴대폰은 핵심 증거입니다. 진정인이 자발적으로 제출하는 경우에도, 조사 과정에서 확보하는 경우에도 올바른 절차가 중요합니다.

포렌식팀이 하는 일
1

기기에서 데이터를 추출

포렌식 전용 도구로 기기의 데이터를 이미지 파일로 추출합니다. 사진, 문자, 앱 데이터 등이 확보됩니다.

2

흔적과 주변 정보로 간접 입증

삭제된 데이터 자체를 되살리는 것은 물리 이미징이 아닌 이상 쉽지 않습니다. 대신 앱에 남은 아티팩트 — 카카오톡 대화 DB, 통화 기록, 위치 로그, 브라우저 기록 등 — 을 조합해 사건을 간접적으로 입증합니다.

흔적 조합 = 간접 입증
3

타임라인 재구성

여러 앱의 기록을 시간순으로 엮어 타임라인을 만듭니다. 근로시간 입증, 부당해고 전후 정황 파악, 임금체불 관련 지시 내역 확인 등에 활용됩니다.

노동사건에서 특히 중요한 점
💡

진정인이 자발적으로 증거를 제출하는 경우

임금체불·부당해고 등 노동사건에서 진정인이 유리한 증거를 직접 제출하는 경우가 많습니다. 카카오톡 대화, 근무시간 기록 앱, 급여 관련 문자 등이 담긴 휴대폰을 포렌식팀에 제출하면, 스크린샷보다 훨씬 강력한 원본 증거를 확보할 수 있습니다.

원본 제출 > 스크린샷 진정인에게 안내 가능
의뢰 전 주의사항
!

기기를 계속 사용하면 흔적이 사라집니다

앱을 실행하거나 메시지를 확인하는 것만으로도 아티팩트가 갱신되거나 덮어쓰여질 수 있습니다. 증거가 필요한 기기는 즉시 사용을 중단하고 포렌식팀에 인계하세요.

사용 중단 후 즉시 인계
!

예전에 쓰던 폰도 제출하세요

사건 당시 사용하던 이전 기기에 핵심 증거가 남아 있는 경우가 많습니다. 기종을 바꿨더라도 예전 폰이 있다면 반드시 함께 제출해 주세요.

구형 기기도 분석 가능
!

파손된 기기도 포기하지 마세요

화면이 깨지거나 전원이 안 들어오는 기기도 내부 저장 칩에서 데이터를 추출할 수 있는 경우가 있습니다. 상태와 무관하게 일단 포렌식팀에 문의하세요.

파손 기기도 일단 문의
!

화면 잠금 해제 정보를 함께 전달

PIN, 패턴, 비밀번호 등 잠금 해제 정보가 있으면 분석이 훨씬 빠르고 범위가 넓어집니다. 잠금이 걸린 기기도 의뢰 가능하지만 분석 가능 범위가 제한될 수 있습니다.

잠금 해제 정보 함께 전달
!

공장 초기화는 절대 금지

기기를 공장 초기화하면 대부분의 아티팩트가 삭제되어 분석이 극히 어려워집니다.

초기화 절대 금지
!

클라우드 데이터는 별도의 법적 절차가 필요

클라우드(iCloud, 구글 드라이브 등)에 저장된 데이터는 기기 분석만으로 확보할 수 없습니다. 압수수색 영장에 클라우드 계정을 명시하는 등 별도의 법적 절차가 필요하므로 사전에 포렌식팀과 협의하세요.

영장에 클라우드 명시 필요 사전 협의 권장
핵심 요약
📋

원본 제출이 최선

스크린샷보다 기기 원본을 제출하면 훨씬 강력한 증거를 확보할 수 있습니다.

📱

구형·파손 기기도 가능

예전 폰이나 파손 기기도 분석 가능합니다. 일단 포렌식팀에 문의하세요.

🔐

잠금 정보도 중요

잠금 해제 정보가 있으면 분석 범위와 속도가 크게 달라집니다.

☁️

클라우드는 별도 절차

영장에 클라우드 계정을 명시하는 등 사전 협의가 필요합니다.

확인 퀴즈
Q. 임금체불 진정 사건에서 진정인이 증거가 담긴 휴대폰을 제출하겠다고 합니다. 가장 적절한 안내는?

⚠ 진정인이 자발적으로 제출하는 경우에도 동의서를 받아두세요. 타인의 기기를 분석하려면 법적 근거(영장 등)가 필요합니다. 의뢰 전 포렌식팀과 절차를 먼저 확인하세요.

🎙️ SoundWriter — 음성 분석 도구

녹취파일을 넣으면 AI가 텍스트로 변환하고, 키워드 검색·하이라이트·번역까지 한 번에. 포렌식 의뢰 없이도 수사관이 직접 녹취록을 분석할 수 있습니다.

시연: 녹취파일 하나로 이 모든 것이 됩니다
1

음성 파일 불러오기 → 자동 텍스트 변환

mp3, wav, m4a 등 음성 파일을 드래그하거나 불러오면 STT AI가 발화 내용을 타임스탬프와 함께 텍스트로 변환합니다. 인터넷 연결 없이 수사관 PC에서 바로 처리됩니다.

오프라인 동작 — 보안 걱정 없음 수십 분 녹취도 수 분 내 처리
2

타임스탬프 클릭 = 해당 시점 즉시 재생

변환된 녹취록의 시간을 클릭하면 해당 시점의 음성이 바로 재생됩니다. 텍스트를 보면서 원본 음성을 동시에 확인할 수 있어, 오인식 여부를 현장에서 즉시 검증할 수 있습니다.

듣고 싶은 부분만 골라 듣기
3

화자 분리 — 누가 무슨 말을 했는지

화자 분리 기능을 켜면 대화 참여자를 자동으로 구분합니다. "피의자가 ~라고 말했다"는 진술의 근거를 녹취록에서 직접 확인할 수 있습니다.

수사관을 위한 핵심 기능
🔍

키워드 검색 + 색상 하이라이트

쉼표로 구분해 여러 키워드를 동시에 검색하면, 각 키워드가 다른 색상으로 하이라이트됩니다. 예를 들어 "임금,해고,협박"을 검색하면 노란색·초록색·파란색으로 구분되어 한눈에 핵심 발언을 파악할 수 있습니다.

최대 6색 자동 배정 수백 줄에서도 핵심만 골라보기
📝

주석 메모 삽입

녹취록 중간에 수사관의 메모를 삽입할 수 있습니다. "여기서 피의자 목소리 톤 변화", "진정인 진술과 일치" 등의 주석을 타임라인에 직접 기록할 수 있습니다.

📁

내보내기 — TXT · PDF · HWPX

분석이 끝난 녹취록을 TXT, PDF, HWPX(한글) 형식으로 내보낼 수 있습니다. 현재 파일만, 선택한 파일들만, 또는 전체 일괄 내보내기가 가능해 보고서 작성이 간편합니다.

HWPX 한글 파일 직접 생성 키워드 하이라이트 색상 유지
단축키 — 키보드에서 손 떼지 않고 분석

녹취록을 읽으면서 음성을 제어하고, 키워드를 탐색하고, 편집까지 키보드만으로 할 수 있습니다.
